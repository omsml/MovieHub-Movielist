# 🎬 MovieHub – Ultimate Movie Listing & Rating Platform

![MovieHub Banner](https://images.unsplash.com/photo-1517604931442-7e0c8ed2963c?auto=format&fit=crop&q=80&w=1000)

**MovieHub** is a modern movie discovery platform that allows users to explore movies across different industries including **Hollywood, Bollywood, Odia, and South Indian cinema**.  

The platform features a **Netflix-style UI**, real-time movie data integration, interactive star ratings, and embedded movie trailers.

---

# 🚀 Key Features

## 📡 Real-Time Data Integration
- Integrated with **TMDB (The Movie Database) API**
- Fetches **real movie titles, posters, and metadata**
- Automatically loads **30+ movies per category**
- Categories include:
  - Trending Movies
  - Hollywood
  - Bollywood
  - South Indian
  - Odia
  - Bangla
  - Horror
  - Drama

---

## 🎥 Trailer Playback
- Watch **official movie trailers inside the website**
- Embedded **YouTube video player**
- Smart fallback system for missing trailers
- "Watch on YouTube" button for restricted videos

---

## ⭐ Interactive Movie Rating System
- 5-star **interactive rating system**
- Hover and click animations
- Ratings stored using **Local Storage**
- Ratings remain saved after page refresh

---

## 🎭 Movie Details Modal
Clicking a movie card opens a **modal popup** that shows:

- Movie Poster
- Title
- Genre
- Release Year
- Description
- Cast Members
- Rating System
- Trailer Player

---

## 📱 Fully Responsive Design

MovieHub is optimized for:

- 📱 Mobile Devices
- 📱 Tablets
- 💻 Laptops
- 🖥 Desktop Screens

Features include:

- Glassmorphism UI
- Smooth animations
- Netflix-style movie sliders
- Horizontal scroll navigation
